## What I Learned

1. [Insight tentang process / RTCC-O] : Saya belajar bahwa kerangka kerja RTCC-O (Role, Task, Context, Constraints, Output) sangat krusial dalam kolaborasi
   teknis. Dengan menetapkan batasan yang ketat—seperti larangan penggunaan tag <div>—saya dipaksa untuk lebih memahami fungsi mendalam dari elemen
   Semantic HTML5

2. [Insight tentang semantic HTML / CSS] : Membangun struktur tanpa <div> membuat kode saya jauh lebih bersih

3. [Insight tentang collaboration dengan AI] : Kolaborasi dengan ai membuat saya lebih cepat dalam menyelesaikan projek. Walaupun AI dapat membantu
   mengerjakan projek lebih cepat, saya belajar bahwa AI dapat membuat kesalahan sehingga saya harus mereview code yang telah dibuat oleh AI untuk 
   memastikan code yang telah dibuat sudah benar atau belum

## Challenges & Solutions

Challenge 1: Layout awal menggunakan sistem vertical stack yang membuat halaman terlalu panjang dan menyisakan banyak ruang kosong di layar desktop, 
sehingga kurang efisien secara visual
How I Solved: Saya mengganti strategi layout menjadi Adaptive Dual-Column Grid. Dengan menggunakan CSS Grid dan Media Queries, 
saya mengatur agar tampilan secara otomatis berubah dari satu kolom menjadi dua kolom sejajar saat dideteksi lebar layar minimal 768px.
Hal ini menciptakan keseimbangan visual yang lebih baik dan profesional
