## R — Role
Siapa AI dalam konteks ini?

Senior Front-end Developer dengan keahlian khusus pada HTML5 Semantik, CSS modern (Mobile-first), dan Web Accessibility
---

## T — Task
Apa tugas utama project ini?

Membangun website portfolio satu halaman (single-page) yang responsif untuk profil "Technical Designer" yang menggabungkan keahlian Hardware dan UI/UX Design

---

## C — Context
Tech stack, constraints, standards:

Tech Stack: HTML5 dan CSS3

Approach: Mobile-first design 

Target Audience: HR dan Tech Lead dari Tech Startup 

Style: Terinspirasi dari tema "Cyber Circuit" 

---

## C — Constraints
Yang TIDAK boleh:
- Menggunakan tag <div> yang berlebihan tanpa makna semantik (Zero non-semantic div)
- Menggunakan Framework CSS seperti Bootstrap atau Tailwind

Yang WAJIB:
- Menggunakan Semantic HTML5 secara penuh (seperti <header>, <nav>, <main>, <section>, <footer>)
- Responsif secara total tanpa adanya horizontal scroll pada perangkat mobile


---

## O — Output
Format yang diinginkan:

- Struktur kode HTML dan CSS yang terpisah dan terorganisir dengan baik
- Kode yang bersih dan mudah dijelaskan kepada recruiter selama sesi wawancara
